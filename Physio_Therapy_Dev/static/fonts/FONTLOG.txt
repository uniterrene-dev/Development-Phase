  Aileron Thin Italic
  --------------------------------------------

  This file provides detailed information on the Aileron font 
  Software.
  This information should be distributed along with the 
  Aileron font and any derivative works.
  
  Aileron Font Information
  
  Aileron is a neogrotesque typeface made by Sora Sagano. The 
  design is close to Helvetica, but conceptually it has 
  become rather close to the Univers. Based upon the multiple 
  master principle, available weights include eight steps 
  from UltraLight to Black.
  
  ChangeLog
  
  22 Jan 2015 (Christoph Haag)
- adjusted OS/2 weight class
  
  9 Jan 2015 (Christoph Haag) Aileron 1.103
- created fontforge sources as base to work on GNU/Linux
- added FONTLOG to .sfdir
- Mastered Font from Fontforge SFDIR to 
  UFO,OTF,TTF,EOT,WOFF,SVG
